## 3. 商業模式

### 變現策略：Freemium + 內容訂閱

**免費層（鉤子）：**
- 12星座完整性格分析（已有）
- 星座相容度（免費限制：每日3次）
- 星座首頁與SEO流量

**付費層（Premium — $120 TWD/月 或 $999 TWD/年）：**
- 每日運勢（每天更新，含愛情/事業/財運/健康四维）
- 每週/月運勢（每週一/每月1日更新）
- 塔羅占卜（78張牌，含牌義解讀）
- 個人化命盤（輸入生日→上升/太陽/月亮星座）
- 進階相容度分析（完整合盤敘述，無限制）
- 年度星座預言（年度專題）

### 流量變現（中期）：
- 聯盟行銷（占星課程、身心靈產品廣告）
- 品牌置入

## 5. 功能藍圖（v2）

### Phase 1：核心內容充實
1. **每日/每週/每月運勢 API 串接**
   - 使用占星邏輯（太陽行運 + 星座能量）生成文字
   - 四维评分：愛情❤️ / 事業💼 / 財運💰 / 健康🏥
   - 每12星座 × 4维 = 48種每日內容
   
2. **塔羅占卜系統**
   - 78張萊德·偉特塔羅牌
   - 三牌陣（過去/現在/未來）或單張占卜
   - 每日免費1次，付費無限

3. **個人化命盤（免費功能）**
   - 輸入：出生日期 + 出生時間（可選）+ 出生地
   - 輸出：上升星座、太陽星座、月亮星座
   - 解釋三大行星在各自星座的含義

### Phase 2：商業化基礎
4. **訂閱系統整合**
   - Stripe 付費牆（免費攔截/功能鎖定）
   - 建議用 Supabase + Stripe 或 Vercel Blob + Stripe
   - 免費試用7天

5. **SEO 全面優化**
   - 每個星座詳細頁獨立的 meta description + OG tag
   - Sitemap 生成
   - 結構化資料（JSON-LD）

### Phase 3：差異化功能
6. **愛情配對雷達圖**
   - 以雷達圖視覺化兩星座在5個維度的相容度
   - 維度：溝通、信任、情感、價值觀、性生活

7. **月度星座專題**
   - 每月一個主題（如「四月牡羊座：開創你的事業新版圖」）
   - 長文深度內容（付費專區）

8. **社群功能（可選）**
   - 分享命盤截圖到社群媒體
   - 星座人格測驗（病毒傳播）

## 6. 付費牆策略

### 免費攔截點設計
```
首頁 → 星座詳細頁 → 點擊「查看完整性格分析」（免費）
                          ↓
              點擊「查看每日運勢」（付費牆）
                          ↓
              訂閱彈窗 → Stripe 付款
```

### 訂閱流程
1. 點擊付費功能 → 彈出「加入 Premium」視窗
2. 選擇月/年方案 → Stripe Checkout
3. 付款成功後設定 Cookie/Session 解除功能鎖
4. 取消訂閱：Stripe Portal

## 7. 技術架構建議

### 現有棧（保留）
- Next.js 16 (App Router) + TypeScript
- CSS Modules（已有）
- Vercel 部署

### 需要新增
| 用途 | 方案 |
|------|------|
| 訂閱支付 | Stripe（需商業驗證） |
| 會員系統 | Supabase Auth（匿名登入 → 信箱登入） |
| CMS/內容管理 | Notion API（文章內容） |
| 每日運勢生成 | 占星邏輯 + OpenAI（可選） |
| 圖片/CDN | Vercel Blob 或 Cloudinary |

### 部署結構
```
Vercel (Next.js)
  ├── / (首頁)
  ├── /[sign] (星座詳細頁 - SEO)
  ├── /compatibility (相容度計算機)
  ├── /tarot (塔羅占卜)
  ├── /horoscope (每日/每週運勢 - 付費)
  ├── /chart (個人化命盤)
  └── /subscribe (訂閱頁)
```
